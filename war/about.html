Draw Diagrams Online, free.

Works Everywhere

draw.io is an online diagramming application and Github project. It features the full range of visual configuration you expect, as well as web application features such as a full range of export options, a large collection of icons, real-time collaboration and embedded widget sharing.

Multiple Integrations

draw.io optionally integrates fully with Google Drive to use your existing Google account cloud storage. There's also a plugin for confluence®, with source code. There's a GWT wrapper with a recreation of draw.io entirely in GWT. We're integrated with Google Apps for Business, with a JIRA plugin coming shortly.

Full Source Code

The full source code to both draw.io and the underlying mxGraph library is available on Github. You can deploy on your own servers, for full data control. There is technical support at the JGraph Q&A site, as well as documentation and videos.
